# Machine-Learning-and-Deep-Learning-Model

## 📌 프로젝트 개요

이 저장소는 다양한 도메인에서 활용 가능한  
**머신러닝 및 딥러닝 기반 모델을 주제별로 정리한 아카이브**입니다.

- 이미지 기반 분석
- 사용자 행동 기반 추천 시스템
- 시계열 및 수요 예측
- 의료 영상 분석
- 재사용/재활용 판단 모델 등
